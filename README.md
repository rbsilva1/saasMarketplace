### A SAAS e-commerce

# How to initialize

```typescript
  npm i // To install all depedencies
  npx prisma db push // To push schema model to your personal database
```

# Techniques

- Singleton
- Repository Pattern
- Single Responsibility Principle (SRP)
- Express
- Dependency Injection

# Tools

- Cookies and JWT for authentication
- Prisma ORM
- Express.Js
- Jest 
- Bcrypt
- Cors
- Nodemon
- Axios to ASAAS API
